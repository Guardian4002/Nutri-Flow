# NutriFlow v2 — Health & Meal Planner

AI-powered nutrition and meal planning app. Works locally with no server needed.

## Features
- BMI & BMR calculator
- Personalized daily & weekly meal plans
- 94 dishes across 8 cuisines (Malaysian, Pakistani, Indian, French, Chinese, Afghan & more)
- Grocery list & cost breakdown in RM
- Works without AI using a local or default dataset
- Supports Anthropic Claude & Google Gemini API keys

## Usage
Download `nutrition-bot.html` and open it in any browser.
Adding an AI api key is optional for better output
