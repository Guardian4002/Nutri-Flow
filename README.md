# NutriFlow v.8.1 — Health & Meal Planner

AI-powered nutrition and meal planning chatbot. Works locally with no server needed.

## Features

- BMI & BMR calculator with full health analysis
- Personalised daily & weekly meal plans
- 94 dishes across 8 cuisines (Malaysian, Pakistani, Indian, French, Chinese, Afghan & more)
- Dietary filters — Halal, Vegetarian, Vegan, High Protein, Low Carb, Gluten-Free, Dairy-Free
- Allergy constraint support
- Grocery list & cost breakdown in RM
- Budget tiers — Hawker, Home Cook, Restaurant
- Meal swap — replace any meal without regenerating the full plan
- 10 evidence-based health tips personalised to your stats
- Calorie reference table across proteins, carbs, fats & beverages
- Works without AI using the built-in dataset
- Supports Anthropic Claude & Google Gemini API keys
- 5 theme presets with full custom colour editor (Dark, Ocean, Forest, Sunset, Light)
- Print support for meal plans
- No sign-up, no server, no data collection — fully client-side

## Usage

1. Download `nutrition-bot.html`
2. Open it in any modern browser
3. Follow the chat to get your personalised meal plan

> Adding an AI API key is optional but recommended for plans with uncommon dietary combinations.

## API Key Setup

Click the key icon (⚙) in the top bar and enter either:
- **Anthropic Claude** key — starts with `sk-ant-...`
- **Google Gemini** key — starts with `AIzaSy...`

Your key is saved locally in your browser and never sent to any external server.

## Dataset

Meal data is fetched from the default GitHub-hosted dataset. You can override it with your own dataset URL in the API settings modal.
